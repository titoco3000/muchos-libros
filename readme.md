# Muchos-libros

IA para recomendação de livros, baseada em uma lista de JSONs.

## Instalação

Crie um arquivo `.env`, baseado em `.env.example`

Inicie um virtual enviroment e instale `requirements.txt`

Para iniciar o banco de dados vetorial:
    
    python3 setup.py

## Uso

    python3 main.py
